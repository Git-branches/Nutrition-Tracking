# Nutrition Tracking #

A web-based system that helps users log their meals, track daily nutrients, and monitor overall diet to support a healthier lifestyle.

# Features
- Log daily meals and snacks
- Track calories, proteins, carbs, and fats
- Monitor progress with reports
- Simple and user-friendly interface

# Tech Stack
- PHP
- MySQL
- HTML, CSS, Bootstrap
- JavaScript
